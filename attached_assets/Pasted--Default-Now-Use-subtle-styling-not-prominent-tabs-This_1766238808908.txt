## 3. TSLE Chart Behavior by Horizon

### Horizon: NOW
- Use existing TSLE time-series
- High-frequency points
- Show regime state (e.g. Stable / Fragile / Strengthening)

### Horizon: SESSION
- Aggregate PoLi values over session window
- Downsample to fewer points
- Smooth transitions
- Show directional liquidity drift

### Horizon: BASELINE
- Replace chart with:
  - Baseline PoLi value
  - Current PoLi vs baseline delta
- Optional minimal sparkline (very low emphasis)

Example UI text:
> “Current liquidity is **+6 points above baseline** — unusually strong.”

- **Depth Δ (%)**
- **Confidence (%)**

Velocity should naturally approach zero in Baseline mode.

Add regime labeling above or beside the TSLE graph:

Examples:
- “Stable — Liquidity holding above 25–50 bps”
- “Fragile — PoLi declining while price stable”
- “Strengthening — Two-sided depth improving”

### Fragility Trigger Rule (UI-only logic):
Display a warning state when:
- PoLi decreases materially
- AND price remains stable
- AND depth imbalance worsens

Tooltip copy:
> “Liquidity is deteriorating beneath a stable price — execution risk rising.”

- No animations that imply trading or price action
